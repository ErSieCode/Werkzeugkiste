GPS-Berechtigung erforderlich

⚠️ Sicherheitshinweis: GPS funktioniert nur über HTTPS-Verbindungen. Bitte verwenden Sie eine sichere Verbindung.

Diese App benötigt Zugriff auf Ihren Standort, um Angelplätze zu markieren und Ihre Position auf der Karte anzuzeigen. Ihre Standortdaten werden nur lokal gespeichert und nicht an externe Server übertragen.

So aktivieren Sie GPS:

  1. Klicken Sie auf "GPS erlauben"
  2. Wählen Sie "Erlauben" im Browser-Dialog
  3. Falls kein Dialog erscheint, prüfen Sie die Adressleiste
  4. Bei Problemen: Einstellungen → Standort → Chrome → Erlauben

GPS-Problembehebung

1. Browser-Berechtigungen prüfen

Klicken Sie auf das Schloss-Symbol in der Adressleiste und stellen Sie sicher, dass "Standort" auf "Erlauben" gesetzt ist.

2. Android-Einstellungen

Gehen Sie zu: Einstellungen → Standort → App-Berechtigungen → Chrome → Erlauben

3. GPS aktivieren

Stellen Sie sicher, dass GPS in den Android-Einstellungen aktiviert ist: Einstellungen → Standort → Ein

4. Browser-Cache leeren

Leeren Sie den Browser-Cache: Chrome → Einstellungen → Datenschutz → Browserdaten löschen

5. Seite neu laden

Laden Sie die Seite neu und versuchen Sie es erneut.
